fastapi_app package¶

This module defines web services that compute FizzBuzz sequence. To run a server instance for manual testing use script run_fastapi_server.sh located in project root folder. Then, in a web browser, open http://localhost:8000/docs or http://localhost:8000/redoc to see available services.
