fastapi_app.constants module¶

This module define constants that are common to fizzbuzz_cli and fastapi_app.

CLASSIC_FIZZBUZZ_PREFIX = 'classic-fizzbuzz'¶

The prefix used for the classic FizzBuzz web services

N_ARGS_NAME = 'n'¶

The name of the integer parameter in the classic FizzBuzz web services

SEP_ARGS_NAME = 'sep'¶

The name of the separator parameter in the classic FizzBuzz web services

STREAM_PATH = 'stream/'¶

The path of the streamed classic FizzBuzz web service
